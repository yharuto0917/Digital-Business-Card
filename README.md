## App Overview

This application is a web-based interactive digital business card that presents profile details and contact information in a single polished interface. The card tilts in response to mouse movement and touch input, and flips between front and back on click or tap, creating a more memorable self-introduction experience than a static profile card.

The visual background uses shader-based rendering built with `Three.js` and `React Three Fiber`, combined with smooth tilt and flip animations powered by `Framer Motion`. The UI is built with `React`, `TypeScript`, and `Vite`, styled with `Tailwind CSS`, and prepared for deployment to `Cloudflare Pages` through `GitHub Actions`.

### Key Features

- 3D card interaction that responds to hover and touch movement
- Flip-card UI with front and back faces
- Custom shader-based background for a refined visual texture
- Compact information design suited for portfolio and profile sharing
- Automated deployment workflow with GitHub Actions and Cloudflare Pages

Some parts of this application are generated by Gemini 3.1 Pro in Google AI Studio.
